## MICROSERVICE TEST

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## Docker

```bash
# up to Docker
$ docker-compose up --build -d
```

## Restore database

Copy sql from setup.sql file in project root and restore database
